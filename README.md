# mobile-app 

Um aplicativo de flashcard para palavras em inglês com o objetivo de aumentar o vocabulário. O usuário terá um perfil, em que poderá criar flashcard de categorias que desejar. Como exemplo, ele poderá criar uma categoria de cumprimentos, e outra para restaurante. Em sua primeira categoria, cumprimentos, poderá inserir flashcards de de palavras em português que ele queira, e o sistema irá inserir um flashcard com a tradução.

Obs.: O aplicativo serviria para aumento de vocabulário, ou seja, palavras ou frases curtas.
